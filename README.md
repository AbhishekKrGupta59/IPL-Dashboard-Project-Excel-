🏏 IPL Dashboard Project (Excel)
📌 Project Overview

This project is an interactive IPL Dashboard created in Microsoft Excel to analyze and visualize Indian Premier League (IPL) data. The dashboard provides meaningful insights into team performance, player statistics, match outcomes, and tournament trends through dynamic charts and KPIs.

🎯 Objectives
Analyze IPL data efficiently.
Create an interactive and user-friendly dashboard.
Visualize key performance metrics using charts and KPIs.
Enable quick decision-making through data-driven insights.
